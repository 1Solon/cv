## Build (Typst)

You can compile the Typst CV to PDF with the Typst CLI:

```powershell
typst compile "main.typ" "docs/saulburgess_cv.pdf"
```

## Editing

- Recommended editor: VS Code with the "Tinymist Typst" extension for live preview and diagnostics.
- Main file: `main.typ`.
